<h1 align="center">Countdown Timer</h1>

<p align="center">
  Esse repositório é destinado ao desafio de criação de um countdown timer, da Rocketseat!
  <br>
  O objetivo é criar uma página de coming soon para o lançamento de um site, produto ou serviço. A página deve conter um countdown timer informando quanto tempo falta para o lançamento. Além disso, ao clicar em "inscreva-se", deve aparecer um modal com um formulário para preenchimento.
</p>

## 💻 Sobre o projeto

Este projeto foi desenvolvido para um dos desafios do curso **Discover**, oferecido pela [Rocketseat](https://blog.rocketseat.com.br/primeira-next-level-week/), que é uma jornada prática de introdução à tecnologia para conhecer o universo da programação.
<br>
<br>
O desafio consistia em criar uma página com uma contagem regressiva, além de um modal para preenchimento de um formulário.
